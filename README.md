# Kyora

Kyora is an AI-native observability and optimization suite for Solana validators.
Built by operators, for operators.

## Installation

```bash
bun add -g kyora
```

## Development

This project uses Bun for development:

```bash
# Install dependencies
bun install

# Run in development
bun dev

# Build for npm
bun run build

# Publish to npm
npm publish
```
